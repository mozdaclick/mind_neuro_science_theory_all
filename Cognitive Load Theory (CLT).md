**Cognitive Load Theory (CLT)** (note: the correct spelling is "cognitive," not "cognetive") is a well-established framework in educational psychology that explains **how humans process and learn new information**, with a strong focus on the limitations of our working memory.

### Origin and Core Idea
- Developed in the 1980s by Australian psychologist **John Sweller**.
- It is based on what we know about human cognition, particularly George Miller’s earlier work on working memory capacity (often summarized as the "magical number seven, plus or minus two" — we can only hold and manipulate a limited amount of information at once).

The central insight: **Learning is most effective when cognitive load (the total mental effort being used in working memory) is managed carefully**. If the load is too high, learning suffers or fails entirely.

### Three Types of Cognitive Load

CLT breaks cognitive load into three categories that add up to the total load on working memory:

1. **Intrinsic Cognitive Load**  
   - The inherent difficulty or complexity of the material itself.  
   - This depends on the **element interactivity** — how many things the learner must hold in mind simultaneously.  
   - Example: Learning basic addition (low intrinsic load) vs. solving complex calculus problems (high intrinsic load).  
   - **This load cannot be changed easily**; it depends on the learner’s prior knowledge. Experts experience lower intrinsic load on the same topic than novices.

2. **Extraneous Cognitive Load**  
   - Unnecessary mental effort caused by **poor instructional design**.  
   - This is the type instructors and designers have the most control over.  
   - Examples:  
     - Slides with too much text and irrelevant images ("split-attention effect").  
     - Unclear explanations.  
     - Searching for information instead of being given it directly.  
   - Goal: **Minimize extraneous load** so more capacity is available for actual learning.

3. **Germane Cognitive Load**  
   - The "good" load — mental effort devoted to **deep processing**, schema construction, and long-term learning.  
   - This is the effort used to organize new information, connect it to existing knowledge, and build usable mental models (schemas).  
   - High germane load is desirable for learning.

**Total Cognitive Load = Intrinsic + Extraneous + Germane**  
Working memory has limited capacity, so if intrinsic + extraneous is already too high, there’s little room left for germane load → poor learning.

### Key Principles and Effects from CLT

Several well-researched effects guide better teaching and design:

- **Worked Example Effect**: Studying fully solved examples is often better for novices than solving problems themselves (reduces extraneous load).
- **Split-Attention Effect**: Separate sources of information (e.g., diagram + text on different pages) should be integrated.
- **Redundancy Effect**: Identical information presented in multiple forms (text + narration saying the same thing) can hurt learning.
- **Modality Effect**: Using both visual and auditory channels (e.g., diagrams + spoken explanation) is better than overloading one channel.
- **Expertise Reversal Effect**: Techniques that help beginners (e.g., lots of guidance) can hinder experts, who benefit from more problem-solving.

### Practical Applications

- **Instructional design** (e-learning, textbooks, lectures): Keep presentations clean, use visuals wisely, provide scaffolding for novices.
- **User Interface / UX design**: Reduce mental effort needed to use apps and websites.
- **Training & Education**: Break complex topics into smaller chunks, use pre-training, provide clear examples before practice.
- **Multimedia learning**: Closely related to Richard Mayer’s Cognitive Theory of Multimedia Learning, which builds on CLT.

### Evolution of the Theory
Originally focused heavily on minimizing extraneous load, later versions (including Sweller’s own updates) emphasize that some forms of "desirable difficulty" can be beneficial if they support schema building. The theory continues to evolve with new research on motivation, emotion, and long-term memory.

### Simple Analogy
Think of working memory as a **small desk** in your brain.  
- Intrinsic load = the size and complexity of the project you *must* put on the desk.  
- Extraneous load = clutter, bad lighting, and distractions on the desk.  
- Germane load = the actual thinking and organizing work that turns the project into useful knowledge you can store in the "filing cabinet" (long-term memory).

If the desk is overloaded with clutter, you can’t do the real work.
